# Gamerpedia

## Descripción

Gamerpedia es un sitio web informativo desarrollado con HTML y CSS cuyo objetivo es ofrecer información sobre el mundo de los videojuegos.

El sitio incluye contenido sobre diferentes videojuegos, géneros, consolas y otros temas relacionados con la industria del videojuego.

Este proyecto fue realizado como práctica del ciclo formativo de Desarrollo de Aplicaciones Multiplataforma (DAM).

---

## Tecnologías utilizadas

- HTML5
- CSS3

---

## Características

- Diseño estático e informativo.
- Navegación entre distintas secciones.
- Organización del contenido por categorías.
- Diseño basado en HTML y CSS.
